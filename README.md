# Docling Java API
A Java API wrapper for IBM's Docling document processing library.
Developed through the Rensselaer Center for Open Source (RCOS) 
in collaboration with an IBM engineer.

## Status
Under active development.

## About
Docling is a powerful AI-powered document parsing library developed by IBM Research.
It is written natively in Python and currently lacks official APIs for other languages.
This project aims to provide a native Java API for Docling.
